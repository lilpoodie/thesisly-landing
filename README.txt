
Thesisly Landing Page - Deployment Instructions
===============================================

Step 1: Upload to GitHub
------------------------
1. Go to https://github.com and log into your account.
2. Click the '+' icon in the top right and select 'New repository'.
3. Name your repo (e.g., thesisly-landing).
4. Click 'Create repository'.
5. On the next page, scroll down to 'â€¦or upload files'.
6. Drag and drop the contents of this unzipped folder (all files and folders inside) into GitHub's upload area.
7. Scroll down, add a commit message like 'Initial commit', and click 'Commit changes'.

Step 2: Deploy on Vercel
------------------------
1. Go to https://vercel.com and log into your account.
2. Click 'New Project'.
3. Select 'Import Git Repository'.
4. Choose the repository you just created.
5. Vercel should detect this is a Next.js app.
6. Click 'Deploy'.
7. Wait for deployment to finish and open your live site URL.

Done!
-----
You now have a fully functional Thesisly landing page with proposal generator API.
